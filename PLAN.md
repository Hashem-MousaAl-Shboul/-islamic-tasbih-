# Fix gaps in the Tasbih screen

## What will be fixed

- **Header-to-cards gap**: The rounded bottom corners of the header create visible ivory-colored gaps against the dark green cards section. Will make the header flow seamlessly into the cards area.
- **Cards-to-content gap**: The dark green cards section ends abruptly against the ivory background. Will add a smooth curved transition between them.
- **Excess vertical spacing**: Reduce unnecessary padding/margins between the dhikr display card, counter area, and control buttons to make the layout tighter and more cohesive.
- **Bottom padding gap**: The hardcoded bottom padding may create an awkward gap above the tab bar. Will adjust to fit naturally with the tab bar.
- **Overall tighter layout**: Remove extra spacing in the scroll content, counter section, and control buttons row so everything feels connected without empty spaces.
