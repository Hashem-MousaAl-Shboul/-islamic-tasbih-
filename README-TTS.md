# نظام TTS (Text-to-Speech) للأذكار

تم إضافة نظام تشغيل صوتي متقدم للأذكار والأدعية باستخدام مكتبة `expo-speech`.

## المميزات

### 🎙️ أصوات متعددة للقراء
- **السديس**: صوت هادئ ومتأني (pitch: 0.9, rate: 0.7)
- **الشريم**: صوت متوسط (pitch: 1.0, rate: 0.8) 
- **العفاسي**: صوت واضح ومرتفع (pitch: 1.1, rate: 0.75)

### 📱 واجهة مستخدم متطورة
- أزرار تشغيل/إيقاف لكل ذكر منفرد
- تشغيل مجموعة كاملة من الأذكار (صباح/مساء/بعد الصلاة)
- شريط حالة يظهر الذكر الحالي والقارئ
- مؤشرات بصرية للحالة (تشغيل/توقف/تحميل)

### 🔄 إدارة متقدمة للتشغيل
- تشغيل متتالي للأذكار مع فترات انتظار
- إيقاف فوري للتشغيل الحالي
- منع التداخل بين التشغيلات المختلفة
- حفظ حالة القارئ المختار

## الملفات المضافة

### `utils/ttsService.ts`
خدمة TTS الرئيسية التي تحتوي على:
- فئة `TTSService` لإدارة التشغيل الصوتي
- إعدادات مختلفة لكل قارئ
- دوال مساعدة للاستخدام المباشر

### `hooks/useTTS.ts`
Hook React لإدارة حالة TTS:
- متابعة حالة التشغيل الحالية
- إدارة الأخطاء
- تحديث الواجهة تلقائياً

### `components/TTSStatusBar.tsx`
شريط حالة يظهر:
- النص الحالي قيد التشغيل
- اسم القارئ
- زر إيقاف سريع
- شريط تقدم بصري

## كيفية الاستخدام

### تشغيل ذكر واحد
```typescript
import { playDhikr } from '@/utils/ttsService';

await playDhikr('سبحان الله', 'alafasy');
```

### تشغيل مجموعة أذكار
```typescript
import { playAdhkarCategory } from '@/utils/ttsService';

const texts = ['سبحان الله', 'الحمد لله', 'الله أكبر'];
await playAdhkarCategory(texts, 'sudais');
```

### استخدام Hook في المكونات
```typescript
import { useTTS } from '@/hooks/useTTS';

function MyComponent() {
  const {
    isPlaying,
    currentReciter,
    playDhikr,
    stop,
    setReciter
  } = useTTS();

  return (
    // واجهة المستخدم
  );
}
```

## متطلبات النظام

- Expo SDK 53+
- مكتبة `expo-speech`
- دعم الصوت في الجهاز

## الاختبار

يمكن اختبار النظام من خلال:
1. فتح شاشة الأذكار
2. اختيار قارئ من الأزرار العلوية
3. الضغط على "استمع" لأي ذكر
4. أو اختيار فئة والضغط على "استمع للفئة كاملة"

النظام يعمل على الجوال والويب مع تحسينات خاصة لكل منصة.
